# Superprompt — site premium para __PROJECT_NAME__

## Missão

Transforme este starter em um site de alto nível para cumprir: **__PROJECT_BRIEF__**

## Antes de codificar

1. Leia `PRODUCT.md`, `DESIGN.md` e `QUALITY.md`.
2. Defina público, oferta, objeção principal e CTA.
3. Escolha uma ideia visual central ligada ao negócio; não use estética genérica de startup.
4. Organize a narrativa: promessa → problema → mecanismo/solução → prova real → oferta → CTA.

## Implementação

- Preserve performance, semântica e responsividade do starter.
- Use componentes por seção somente quando isso melhorar manutenção.
- Crie hierarquia editorial forte e variação de composição entre seções.
- Use animação para orientar atenção, sem atrasar leitura ou ação.
- Se não houver provas, logos ou números reais, use estrutura pronta para receber dados; não invente.
- Todo link e botão precisa funcionar.
- Inclua metadados, Open Graph, favicon e conteúdo adequado ao projeto quando os dados existirem.

## Validação

Inspecione o site no navegador em mobile e desktop. Confirme favicon, ausência de overflow horizontal e campos sem autozoom. Depois rode `npm run check`, corrija todos os erros e confira `QUALITY.md` item por item.
